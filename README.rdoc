== README

* This app records and display callbacks details for the  Nexmo SMS APIs:

- Details for the Nexmo SMS APIs:
   - SMS Delivery Receipts (DLRs)
   - Mobile Originated SMS (MOs)
 - Details for the Nexmo Voice APIs:
   - Call API Return Parameters
   - Text-To-Speech API Return Parameters

 - Link to heroku logs

* Admin Dashboard with RailsAdmin
* Admin model
* Authentication with Sorcery


* Ruby version  2.3.6
* Rails version 4.2.10
