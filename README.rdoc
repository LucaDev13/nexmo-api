== README

This app records and display details for the  Nexmo SMS APIs - MS Delivery Receipts (DLRs), Mobile Originated SMS (MOs);
details for the Nexmo Voice APIs - Call API Return Parameters, Text-To-Speech API Return Parameters. There is also a link to
heroku logs.


* Ruby version  2.2.0p0 (2014-12-25 revision 49005) [x86_64-darwin14]
* Rails version 4.2.1
